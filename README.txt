A responsive scroller that allows you to show Steemit posts with a set amount of scrolling items.

== Description ==

The size and amount of scrolling items will automatically adjust depending on the browser and screen size. The module features 6 preset themes and is fully customizable.

== Installation ==

= Instructions =

1. Install the Steemit Scroller plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the `/wp-content/plugins/` directory).
2. Activate the Steemit Scroller plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Steemit Scroller' Scrollers page and create a new Scroller.
4. Use the shortcode `[steemitscroller id="scroller_id"]` in your page, post or widget to display a scroller.
